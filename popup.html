UT Lectures Plus

Experimental includes Summary, Semantic Search, and Study Agent tabs.

Search modes

Keyword: Finds exact words/phrases in captions.

Semantic: Finds conceptually similar parts of the lecture, even with different wording.

Study Agent

Study Agent takes in your question, then finds the relevant context in the lecture to give you the best possible answer.

OpenAI API key

An API key lets the extension call OpenAI for summaries, semantic search embeddings, and Study Agent responses.

Get one at platform.openai.com/api-keys, then save it in the Summary tab.
